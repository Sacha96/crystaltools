
# Mindmap

examples without /

```/mindmap
root
  topic1
    subtopic
  topic2
    subtopic
  topic22
    subtopic
  topic23
    subtopic        
      ssswe
        sssww
    sddsdsd
      dd
```



```/mindmap-json
{
  "name": "root",
  "children": [
    {
      "name": "topic1",
      "children": [{ "name": "subtopic" }]
    },
    {
      "name": "topic2",
      "children": [{ "name": "subtopic" }]
    }
  ]
}
```
